# QUICK START

## gradle kts
```
implementation('me.saro:jwt:2.1.0')
```

## gradle
```
compile 'me.saro:jwt:2.1.0'
```

## maven
``` xml
<dependency>
  <groupId>me.saro</groupId>
  <artifactId>jwt</artifactId>
  <version>2.1.0</version>
</dependency>
```
