# QUICK START

## gradle kts
```
implementation('me.saro:jwt:2.0.0')
```

## gradle
```
compile 'me.saro:jwt:2.0.0'
```

## maven
``` xml
<dependency>
  <groupId>me.saro</groupId>
  <artifactId>jwt</artifactId>
  <version>2.0.0</version>
</dependency>
```
