# QUICK START

## Gradle
```
implementation('me.saro:jwt:3.0.0')
```

## Maven
``` xml
<dependency>
  <groupId>me.saro</groupId>
  <artifactId>jwt</artifactId>
  <version>2.1.0</version>
</dependency>
```
